## 🎯 Overview

This is a **personal portfolio website** designed for a Frontend Developer and UI/UX Designer. The website showcases professional skills, featured projects, and provides a contact form for potential clients and employers. Built with pure HTML5, CSS3, and JavaScript without any external frameworks, it demonstrates mastery of modern frontend technologies.

### Purpose
- Present professional profile and skills
- Showcase portfolio projects with live demos
- Provide easy contact method for potential clients
- Demonstrate frontend development capabilities
- Serve as a living resume for job applications

## ✨ Features

### Design & UI
| Feature | Description |
|---------|-------------|
| 🎨 **Glassmorphism Design** | Modern frosted glass effect with backdrop blur |
| 🌓 **Dark/Light Mode** | Theme toggle with localStorage persistence |
| 🖱️ **Custom Cursor** | Animated circular cursor with hover effects |
| 📱 **Fully Responsive** | Perfect layout on mobile, tablet, and desktop |
| ✨ **Smooth Animations** | Scroll reveals, hover effects, and transitions |
| 🎯 **Typing Animation** | Dynamic role display (Frontend Developer, UI/UX Designer, etc.) |
| 🔄 **Float Animation** | Profile image with smooth floating effect |

### Navigation
| Feature | Description |
|---------|-------------|
| 🧊 **Glass Navigation Bar** | Sticky header with blur effect |
| 🔽 **Scroll Hide/Show** | Navbar hides when scrolling down, appears when scrolling up |
| 📱 **Mobile Menu** | Hamburger menu with smooth overlay animation |
| 🔗 **Active Link Detection** | Automatic highlighting of current section |

### Sections
| Section | Content |
|---------|---------|
| **Hero** | Introduction, typing animation, CTA buttons, social links |
| **About** | Professional background, statistics cards |
| **Skills** | Progress bars, technology tags, categorized skills |
| **Projects** | Featured work with GitHub and demo links |
| **Contact** | Contact form (EmailJS integration), contact information |

### Contact Form
| Feature | Description |
|---------|-------------|
| 📧 **EmailJS Integration** | Sends messages directly to Gmail |
| ✅ **Form Validation** | Client-side validation for all fields |
| 🔄 **Loading State** | Visual feedback during submission |
| 💾 **Local Backup** | Messages saved to localStorage as fallback |
| 🔔 **Toast Notifications** | Success/error messages with animations |
